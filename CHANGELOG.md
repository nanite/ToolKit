### Changed

- Added more copy types for the `/tk copy` command. You can now use the following types. Please note that `plain` is the default if no type is provided. Plain is the old default.

  - `KUBEJS_NATIVE`
  - `JSON`
  - `SNBT`
  - `NBT`
  - `CRAFTTWEAKER`
  - `PLAIN`
  - `CSV`
